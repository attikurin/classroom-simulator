教室レイアウトシミュレーター 🏫

先生のための、直感的で使いやすい教室座席・レイアウト配置シミュレーションツールです。
学級開き、席替え、研究授業、グループ学習の事前計画などに活用できます。

🌟 特徴

直感的な操作: 机、椅子、教卓、掲示物などをドラッグ＆ドロップで自由に配置できます。

自動配置パターン: 「一斉授業（6列）」「グループ学習（4人班）」「コの字型」など、一般的なレイアウトをワンクリックで適用できます。

視認性チェック機能: 座席から黒板やスクリーンへの視線を表示し、角度がきつい座席や遠すぎる座席を色分け（赤・オレンジ・緑）で警告します。

ローカル保存: 作成したレイアウトをブラウザ（LocalStorage）に保存し、後から呼び出すことができます。

レスポンシブ対応: PCでの操作はもちろん、タブレットやスマートフォンでも簡易的な操作が可能です。

インストール不要: HTMLファイル1つで動作するため、インターネット環境がない場所でも動作し、USBメモリ等での共有も簡単です。

🎮 使い方

1. 起動方法

classroom_simulator.html を各種Webブラウザ（Chrome, Edge, Safari, Firefoxなど）で開いてください。

2. 基本操作

アイテム追加: 左側（スマホは下部）のメニューから「机」「椅子」「掲示物」などをクリックしてキャンバスに追加します。

移動: アイテムをドラッグして好きな場所に配置します。

回転: アイテムをダブルクリック（スマホはダブルタップ）すると90度回転します。

削除: アイテムをドラッグして、右下に表示されるゴミ箱アイコンへドロップすると削除されます。「全消去」ボタンでリセットも可能です。

3. 便利機能

視認性確認ボタン: 押すと視認性チェックモードになります。もう一度押すと解除されます。

パターン適用: メニューから自動配置パターンを選択し、人数を入力すると自動で並べ替えます。

保存/読込: 右上のボタンで現在の状態を保存・復元できます。

⚠️ 注意事項

「保存」機能はブラウザの「ローカルストレージ」を使用します。異なるブラウザや端末間ではデータは共有されません。

ブラウザのキャッシュをクリアすると保存データが消える場合があります。
